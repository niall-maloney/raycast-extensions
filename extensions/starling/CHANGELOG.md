# Starling Changelog

## [Initial Release] - 2026-02-18

- Added secure personal-token connection flow (user-owned token only).
- Added Dashboard command for account holder and balances.
- Added Transactions command with search, filters, and note update action.
- Added Spaces command for savings/spending spaces.
- Added Payees command.
- Added Cards command with lock/unlock and control updates.
- Added Direct Debit Mandates command with cancellation action.
- Added robust shared API client, error handling, and formatting utilities.

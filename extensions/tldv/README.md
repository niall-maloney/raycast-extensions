# tl;dv Meetings

Browse and manage your [tl;dv](https://tldv.io) meeting recordings directly from Raycast.

## Features

- Browse and search tl;dv meeting recordings
- View meeting transcripts and AI-generated highlights
- Multiple workspace support (up to 3)
- Filter meetings by date (Today, This Week, This Month)
- Favorite/pin frequently accessed meetings
- Export transcripts in Markdown, TXT, or JSON format
- Quick access to recent meetings via menu bar
- Open meetings directly in tl;dv

## Setup

1. Get your tl;dv API key from [tl;dv API Settings](https://tldv.io/app/settings/api)
2. Open the extension preferences in Raycast
3. Enter your API key in the **Workspace 1 API Key** field

### Multiple Workspaces

You can configure up to 3 workspaces with separate API keys. Set the **Default Workspace** preference to choose which workspace loads by default.

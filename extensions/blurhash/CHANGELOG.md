# BlurHash Changelog

## [Initial Version] - 2026-02-24

- Generate blurhash from clipboard images and videos
- Auto-detect files from clipboard, Finder copy, and Finder selection
- Decoded blurhash preview with file metadata
- Native Finder file picker fallback
- Clipboard History deeplink
- Single-line copy: `filename 1920x1080 16:9 LKO2?U%2Tw=w...`
- Supports JPEG, PNG, GIF, BMP, WebP, TIFF, HEIC, HEIF, AVIF, ICO
- Supports MP4, MOV, AVI, MKV, WebM, M4V, FLV, WMV, 3GP, TS

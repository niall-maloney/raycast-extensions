# PromptNote Changelog

## [Initial Release] - 2026-02-24

### Added

- Search, create, and manage prompt notes from Raycast
- Quick Capture for saving snippets without leaving your workflow
- Hierarchical tag browsing (e.g., `ai/prompts/chatgpt`)
- Favorite snippets view
- Archive with restore functionality
- PIN protection with AES-256-GCM encryption and session-based unlock
- Authentication via Email, GitHub, and Google (OAuth PKCE)
- Open notes directly in the PromptNote web app
- Send Feedback action linking to the FeatureBase board
- Copy snippets to clipboard with keyboard shortcuts
- Rich note list with tag badges, pin indicators, and relative timestamps
